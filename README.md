# Builder-Founder Portfolio

Next.js + TypeScript + Tailwind ile hazirlanmis profesyonel portfoy iskeleti.

## Baslangic

```bash
npm install
npm run dev
```

## Sayfalar
- /
- /about
- /projects
- /tools
- /process
- /contact

## Ozellestirme
- Profil bilgilerini `src/data/profile.ts` dosyasindan guncelle.
- Proje kartlarini `src/data/projects.ts` dosyasindan guncelle.

## Notlar
- Icerik plani: `plan.md`
- Metin taslaklari: `content.md`
- Site yapisi: `sitemap.md`
- Uygulama gorevleri: `tasks.md`
