# Portfolio Build Tasks

## 1) Planning & Setup
- [ ] Domain ve isim kararını netleştir
- [ ] Dil kapsamı seçimi (TR, EN, veya iki dil)
- [ ] Referans portföy örneklerini belirle (3-5 adet)
- [ ] Tech stack kesinleştir (Next.js + TS + Tailwind)

## 2) Content Collection
- [ ] Kısa bio metni finalize et
- [ ] Uzun bio metni finalize et
- [ ] İletişim bilgilerini hazırla (mail, LinkedIn, GitHub)
- [ ] Proje havuzunu çıkar (en az 5 proje)
- [ ] MVP için öne çıkan 3 projeyi seç
- [ ] Her proje için problem/çözüm/stack/sonuç metinlerini yaz
- [ ] Proje görsellerini topla veya üret

## 3) UX & UI
- [ ] Düşük sadakat wireframe oluştur (desktop + mobile)
- [ ] Tasarım yönünü belirle (renk, tipografi, component stili)
- [ ] Ana komponentleri tasarla:
- [ ] Header
- [ ] Hero
- [ ] Project card
- [ ] Tag/chip
- [ ] CTA section
- [ ] Footer
- [ ] Responsive breakpoint kontrolleri yap

## 4) Development (MVP)
- [ ] Proje iskeletini oluştur
- [ ] Route yapısını kur (`/`, `/about`, `/projects`, `/contact`)
- [ ] Ortak layout ve navigasyonu geliştir
- [ ] Home sayfasını geliştir
- [ ] About sayfasını geliştir
- [ ] Projects liste sayfasını geliştir
- [ ] Contact sayfasını geliştir
- [ ] Form validasyonu ekle
- [ ] SEO metadata ekle (title, description, OG)
- [ ] favicon ve temel manifest ayarları

## 5) Data & Content Integration
- [ ] Proje verilerini JSON/MDX ile bağla
- [ ] İçerikleri `content.md` ile eşleştir
- [ ] Eksik alanlar için placeholder yerine gerçek veri gir

## 6) Quality Assurance
- [ ] Mobil uyumluluk testi
- [ ] Tarayıcı testi (Chrome, Safari)
- [ ] Lighthouse performans kontrolü
- [ ] Accessibility hızlı denetim (kontrast, heading sırası, alt text)
- [ ] Kırık link kontrolü

## 7) Launch
- [ ] Deploy ortamını bağla (Vercel)
- [ ] Domain yönlendirmesini yap
- [ ] Analytics ekle (GA4/Plausible)
- [ ] Search Console setup
- [ ] Yayın sonrası hızlı smoke test

## 8) Post-Launch (v2)
- [ ] Project detail sayfaları ekle (`/projects/[slug]`)
- [ ] Tools sayfası ekle
- [ ] Process sayfası ekle
- [ ] Blog/build log ekle
- [ ] Çoklu dil desteği ekle

## 9) Timeline Önerisi
- Hafta 1:
- [ ] İçerik toplama + wireframe
- [ ] Görsel yön belirleme
- Hafta 2:
- [ ] MVP geliştirme
- [ ] Responsive düzeltmeler
- Hafta 3:
- [ ] QA + SEO + yayın

## 10) Definition of Done (MVP)
- [ ] 4 temel sayfa yayında
- [ ] En az 3 proje kartı dolu
- [ ] İletişim kanalları aktif
- [ ] Lighthouse temel skorları kabul edilebilir seviyede
- [ ] Mobilde kritik görünüm problemi yok
